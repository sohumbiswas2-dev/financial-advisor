# 🏛️ The Financial Advisor

A historical financial market simulation and gamified investing laboratory built with **React**, **TypeScript**, and **Tailwind CSS**.

"The Financial Advisor" places players in the shoes of an institutional portfolio manager navigating some of history's most volatile economic epochs. From the Dot-Com Bubble to the 2008 Subprime Crisis, players must allocate capital, react to live market news, manage stakeholder sentiment in the boardroom, and navigate unpredictable "Black Swan" events.

## ✨ Key Features

- 🕰️ **Historical Case Archives:** Play through meticulously structured scenarios based on real historic macroeconomic data (e.g., The 2008 Financial Crisis).
- 📈 **Dynamic Sector Simulation:** Allocate capital across uniquely anonymized companies and watch your portfolio react to monthly dynamic events.
- 🦢 **Black Swan Events:** A custom volatility engine ensures history doesn't always repeat exactly. Expect sudden, randomized macroeconomic shocks.
- 💼 **The Boardroom:** A gamified career mode. Accumulate XP and Reputation spanning multiple play-throughs via LocalStorage, unlock high-level "classified" archive cases, and earn trophies for strategic achievements (e.g., *The Rainmaker*, *Cash is King*).
- 🧠 **Beginner Mode & Economics Glossary:** Complex financial jargon is instantly transformed into plain-English definitions with a toggle, ensuring accessibility for all players.
- 🖨️ **Printable Tear-Sheets:** Seamlessly export an aesthetic PDF summarizing your final portfolio performance against benchmarks.

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sohumbiswas2-dev/financial-advisor.git
   ```
2. Navigate to the project directory:
   ```bash
   cd financial-advisor
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the local development server:
   ```bash
   npm run dev
   ```

The application will launch on your local network (typically port `3000`).

## 🛠️ Technology Stack
- **Frontend Core:** React 19, TypeScript, Vite
- **Styling UI:** Tailwind CSS V4, Custom CSS typography & print parameters
- **Animations:** Motion
- **Data Vis:** Chart.js, React-Chartjs-2
- **Icons:** Lucide-React

## 📄 Notes
This project is for educational and gamification purposes. The historical simulation data parameters are carefully tuned abstracts intended to demonstrate macroeconomic interactions.
